# Skills 技能集合

一个专注于软件工程和设计领域的技能集合项目。

## 项目结构

```
Skills/
├── design-first/                    # 设计优先方法论
│   ├── SKILL.md                     # 技能定义文档
│   └── references/                  # 参考资料
│       ├── 决策框架.md
│       ├── 敏捷开发实践.md
│       └── 软件工程方法论.md
│
├── 前端美学设计技能/                 # 前端设计技能
│   ├── SKILL.md                     # 技能定义文档
│   └── references/                  # 参考资料
│       ├── 技术资源/
│       │   ├── CSS技巧.md
│       │   └── 动画库.md
│       ├── 案例分析/
│       │   └── 优秀界面拆解.md
│       └── 设计灵感/
│           ├── 动效参考.md
│           ├── 字体推荐.md
│           └── 配色方案.md
│
└── git-changelog-generator/         # Git 变更日志生成器
    └── SKILL.md                     # 技能定义文档
```

## 技能说明

### design-first

设计优先方法论技能，包含决策框架、敏捷开发实践和软件工程方法论等核心参考资料。

### 前端美学设计技能

专注于前端视觉设计与用户体验，包含 CSS 技巧、动画库推荐、优秀界面拆解、动效参考、字体推荐和配色方案等设计资源。

### git-changelog-generator

Git 变更日志生成器技能，将技术性的 Git 提交记录转化为清晰、结构化且面向非技术用户的变更日志，帮助最终用户理解产品更新内容。

## 使用方式

各技能目录下均包含 `SKILL.md` 技能定义文档，参考资料位于各技能目录的 `references/` 文件夹中。
